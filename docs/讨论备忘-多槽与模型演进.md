## 1. 当前实现要点

| 能力 | 说明 |
|------|------|
| 多槽推理 | `RunEnabledSlots`：`yolo` → `scrfd` |
| 场景 | `idle`（yolo）、`person`（yolo+scrfd） |
| warm 池 | `DisableSlot` 保留 RKNN，再 Enable 复用 |
| 锁外 Init | `EnableSlot` 在锁外 `Clone+Init` |
| 场景 dwell | `scene_dwell_frames` 稳定后切换 |
| yolo+scrfd 绘制 | 可抑制 YOLO person 框，保留人脸框 |
| 启动 Warmup | `WarmupSlot(scrfd)` |
| LLM | 人脸稳定 → 静态自动问候（banner）；用户输入 → `SubmitPrompt` / RKLLM（可选） |

---

## 2. 历史排障（仍有效）

- **Init 死锁**：`Init` 持锁内调 `EnableSlot` → 已改为锁外 Enable。
- **运行期卡顿**：探针+ppocr 每帧抢 NPU → **已通过移除 OCR 消除**；保留 warm 池与锁外 Init。
- **Ctrl+C**：二次 SIGINT `_exit`；队列 drain。

## 4. 未来模型引入

### 4.1 YOLO-World（视觉）

- 新 adapter 目录 `adapters/yoloworld/` 或替换 `yolo` 槽。
- 用 prompt 匹配置位 `person_present`；与 scrfd、llm 并行。

### 4.2 Whisper（音频）

- 需 `AudioSource`、VAD、独立线程；**不能**复用 vision Pipeline 每帧 `cv::Mat` 路径。
- 新信号如 `speech_present`，扩展 `ModelCoordinator` 或旁路模块。

### 4.3 其它 OCR（若恢复「看字」）

- 新 adapter + 可选 `ITextRegionProbe`；**勿**假设仓库仍含 `adapters/ppocr`。

### 4.4 组合示例

| 模块 | 输入 | 触发 |
|------|------|------|
| YOLO / YOLO-World | 视频 | 常开 / person |
| SCRFD | 视频 | person 稳定 |
| LLM | 文本 | face 稳定 |
| Whisper | 音频 | VAD（待建） |
